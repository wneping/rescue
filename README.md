🐾 尋獲毛小孩任務 (Pet Finder Mission)
這是一個基於 Python Flask 與 Leaflet.js 開發的即時地理資訊系統，旨在建立一個全台通用的「走失毛小孩通報平台」。透過 GPS 自動定位與雲端圖片整合，讓熱心路人能在發現走失動物的第一時間，完成最精準的座標標註與照片回報。

🌟 核心功能 (Key Features)
📍 即時 GPS 精準定位
一鍵定位：開啟網頁即自動獲取使用者目前的 GPS 座標。

自動帶入：系統自動將座標填入發布表單，減少手動輸入誤差。

手動修正：使用者也可透過點擊地圖任意位置，手動校正標註點。

📸 雲端照片通報系統
Cloudinary 整合：採用專業雲端圖片託管，確保在免費伺服器環境下照片依然能永久保存。

即時顯示：地圖標記內嵌毛小孩照片，方便搜救者辨識特徵。

🛡️ 權限與安全管理
免登入通報：路人無需註冊即可發布任務，爭取救援黃金時間。

聯絡電話驗證：強制要求填寫電話，確保飼主能即時聯繫發現者。

管理員後台：設有專屬管理員帳號 (wxp800218)，具備刪除過期或違規資訊的權限。

🛤️ 搜尋路徑追蹤
足跡可視化：地圖上以綠色虛線連接已通報地點，協助搜救人員掌握可能的移動路徑。

留言互動：每個地點標籤均設有留言板，供路人更新毛小孩最新的動向。

🛠️ 技術棧 (Tech Stack)
後端: Python 3.x, Flask

前端: HTML5, CSS3, JavaScript (Vanilla JS)

地圖引擎: Leaflet.js (OpenStreetMap)

圖片託管: Cloudinary API

部署平台: Render / GitHub



🚀 快速開始 (Quick Start)
1. 環境準備
複製此專案並安裝必要套件：

Bash
git clone https://github.com/wneping/pet-mission.git
cd pet-mission
pip install -r requirements.txt


2. 環境變數設定 (Environment Variables)
請確保在系統中或部署平台設定以下變數：

CLOUDINARY_NAME:

CLOUDINARY_API_KEY:

CLOUDINARY_API_SECRET:

FLASK_SECRET_KEY:



3. 本地執行
Bash
python app.py
開啟瀏覽器前往 http://127.0.0.1:5000 即可開始使用。



📖 使用指南
發現毛小孩：開啟網頁並允許 GPS 定位。

填寫資訊：確認地圖標記點正確，填入您的聯絡電話與毛小孩特徵。

上傳照片：直接拍照或選取照片。

發布：點擊「發布救援任務」，全國的使用者皆可在地圖上看見該資訊。

後續追蹤：若有最新動向，可在該標記點下方留言回報。



👤 管理員權限
帳號: wxp800218

功能: 登入後點擊地圖上的標記，可執行「刪除案件」操作，用於清理已尋獲或無效的資訊。
